1.Prerequisites

1. mercurial
2. QT5 development packages (recommended version > 5.4)


Debian/Ubuntu Linux distribution:    

<pre>
 apt-get install mercurial
 apt-get install qt5-default
 </pre>   
     
  
2. Downloading NetAnim (NetAnim 3.108)
hg clone http://code.nsnam.org/netanim 


3. Building and Starting NetAnim

NetAnim uses a QT build tool called qmake. Qt versions from 5.4 are supported. Please read the #Prerequisites before proceeding 

<pre>
cd netanim
make clean
qmake NetAnim.pro
make
</pre>
